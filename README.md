## How to Run Programs

### Running C++ Programs

Most C++ files are standalone programs. You can compile and run any `.cpp` file using a C++ compiler such as `g++`.

Example:

```bash
g++ "c++/BackTracking/redInAMaze.cpp" -o redInAMaze
./redInAMaze
```

On Windows PowerShell:

```powershell
g++ "c++\BackTracking\redInAMaze.cpp" -o redInAMaze.exe
.\redInAMaze.exe
```

### Running Python Notebooks and Scripts

Python practice files are mainly stored as Jupyter notebooks (`.ipynb`) with some scripts (`.py`) and datasets (`.csv`).

To open notebooks:

```bash
jupyter notebook
```

To run a Python script:

```bash
python "python course/code by sir/app.py"
```
