## How to Run Programs

Most files are standalone C++ programs. You can compile and run any `.cpp` file using a C++ compiler such as `g++`.

Example:

```bash
g++ "c++/BackTracking/redInAMaze.cpp" -o redInAMaze
./redInAMaze
```

On Windows PowerShell:

```powershell
g++ "c++\BackTracking\redInAMaze.cpp" -o redInAMaze.exe
.\redInAMaze.exe
```
